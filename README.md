# Tri-Model Smart Justice Framework (TMSJ)

AI-based legal framework for Saudi commercial case analysis using NLP and deep learning techniques.

## Project Overview
This project introduces a multi-model legal AI framework designed for Saudi commercial cases.

The framework consists of:

- Model A: Legal entity extraction using AraBERT-based NER
- Model B: Legal scenario generation using AraGPT2
- Model C: Legal outcome prediction

## Project Components

### Data Collection
Web scraping and legal data collection scripts.

### Data Preprocessing
Data cleaning, normalization, BIO tagging, and entity preparation.

### Model A
Named Entity Recognition (NER) for extracting:
- Case numbers
- Courts
- Dates
- Legal references
- Financial amounts
- Parties

### Model B
Legal scenario generation and structured legal analysis generation.
